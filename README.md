# Netflix_loginpage_clone

A responsive Netflix Login Page Clone built using HTML & CSS.  
This project replicates the modern UI design of Netflix’s login interface for learning and practice purposes.

##  Technologies Used
- HTML5
- CSS3
- Responsive Design (Flexbox)

---

##  Features
- Clean and modern UI
- Fully responsive design
- Background image with overlay effect
- Styled input fields and buttons
- Hover effects

---

##  Project Structure

netflix-login-clone/
│── index.html
│── style.css
│── images/

---

## Purpose
This project was created to improve front-end development skills and understand layout structuring using HTML and CSS.
